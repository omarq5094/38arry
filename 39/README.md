# Real Estate Platform MVP v1

هذه نسخة Frontend أولية مرتبطة بـ Google Apps Script API.

## الملفات

- `index.html`: واجهة العملاء، عرض العقارات، وإضافة عقار للمراجعة.
- `property.html`: تفاصيل العقار وإرسال Lead.
- `admin.html`: لوحة الموظف لمراجعة العقارات وطلبات العملاء.
- `config.js`: رابط API.
- `app.js`: منطق واجهة العملاء.
- `admin.js`: منطق لوحة الموظف.
- `styles.css`: التصميم.

## طريقة التشغيل

1. افتح `index.html` مباشرة للتجربة المحلية.
2. أو ارفع الملفات إلى GitHub Pages / Netlify.
3. تأكد أن رابط API في `config.js` صحيح.

## ملاحظات مهمة

- رفع الصور يستخدم Base64 ويرسلها إلى Apps Script.
- كل صورة أكبر من 5MB سيتم رفضها من الواجهة.
- Token الحالي مؤقت للتجربة وليس Login نهائي.
- في النسخة التالية يفضّل إضافة Login حقيقي و Password Hash و Session Expiry.
